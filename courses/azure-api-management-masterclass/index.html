description

About this course

Note that this course doesn't require any prior experience on API, API Gateway or Azure. Necessary basics are covered in the course. Learn the feature-packed Microsoft Azure API Management Service with this HANDS-ON DRIVEN course.

Azure API Management is Platform-as-a-Service (PaaS) offering from Microsoft which provides low-code unified cloud platform for implementing API Gateway on HTTP services and managing complete lifecycle for the APIs.

This course is carefully structured to make sure it provides a holistic learning journey for ANYONE wanting to explore the numerous configurations, functionalities and use-cases of Azure API Management Service.

JULY-2026: Major Course Update (22 new lessons) to cover latest AI Gateway, Resiliency and Security Features

  • Understand and build your own remote MCP Server from APIs
  • Configure and test MCP Server from AI Agents (Cursor, Claude Code)
  • Build LLM APIs using new LLM import feature (OpenAI, Gemini, Ollama APIs)
  • Improve Resiliency with Load Balancers and Circuit Breakers
  • OAuth2 Security required by Backend API? Learn Credential Manager with Microsoft Graph API
  • API Centre Customization
  • Publishing and exploring API Centre Portal

New to Azure, Cloud or APIs? No worries.

  • Curated for everyone — beginners & experienced
  • Covers the necessary Cloud, API and AI fundamentals in dedicated sections and lectures
  • Explore Azure APIM essential concepts and features, then organically dive into advanced features and implementations (including AI Gateway) with subsequent sections

Familiar with Azure and/or APIs but not with API Management? Somewhat familiar with API Management? GOOD!

  • Now explore with me all the flavours of this feature-packed API Management Service — go from Beginner/Intermediate to EXPERT!
  • To make our APIM Service Implementation and Architecture robust, we'll implement various solutions using other tools and services.

So join me to implement API Management and also explore some other related services along the way:

  • App Registrations, OAuth2 Flows and Azure AD (Microsoft Entra ID) for delegated authentication and security
  • Azure Application Gateway, Front Door, Private Endpoint, VNets, NSGs for APIM Networking
  • GitHub Actions for APIOps Implementation
  • Docker Containers and AKS (Azure Kubernetes Service) for deploying Self-hosted Gateway
  • Azure Management REST API for Operations e.g. backup, restore & recovery options
  • Azure Key Vault, Managed Identity, Custom Domains, DNS, Blob Storage, VMs etc.
  • MCP Servers, OpenAI API, Ollama API, Gemini Interactions API, OpenAI Compatible APIs from Ollama and Gemini

Microsoft Azure is the fastest growing major Cloud Platform and this API Management offering is one of the most popular and widely used API Management solutions for a variety of IT organizations.

I've created this course out of my passion for teaching & technology. Having spent 15 years working in enterprise integration and all its flavours (APIs, ETL, EDI, Streaming, MCP, Agent Skills etc.), this service seemed like a good starting point to bring you some crucial aspects of what I've learnt from project implementations in both Integration and Cloud. Happy Learning!!
